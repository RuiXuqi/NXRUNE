================== Windows ===================

Step 1: Find your DELTARUNE install path.
Step 2: Copy the folder path.
Step 3: Run "batch_apply_xdelta.bat".
Step 4: Paste the folder path.
Step 5: Press Enter.
Step 6: Have fun :D

=============== Linux/SteamOS ================

Step 1: Find your DELTARUNE install path.
Step 2: Copy the folder path.
Step 3: Run "batch_apply_xdelta.sh".
Step 4: Paste the folder path.
Step 5: Press Enter.
Step 6: Have fun :D

==================== NOTE ====================

If you're getting an error like 
"xdelta3: target window checksum mismatch: XD3_INVALID_INPUT"
verify integrity of the game files and try again.

The script backs up the original files to
"backup/nxrune/.../data.win" before replacing them.
