================== Windows ===================

Step 1: Find your DELTARUNE install path.
Step 2: Copy the folder path.
Step 3: Run "batch_apply_csx.bat".
Step 4: Paste the folder path.
Step 5: Press Enter.
Step 6: Have fun :D

=============== Linux/SteamOS ================

Step 1: Find your DELTARUNE install path.
Step 2: Copy the folder path.
Step 3: Run "batch_apply_csx.sh".
Step 4: Paste the folder path.
Step 5: Press Enter.
Step 6: Have fun :D

==================== NOTE ====================

The script backs up the original files to
"backup/nxrune/.../data.win" before replacing them.
